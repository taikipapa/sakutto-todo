# サクッとTodo

サクッとTodoは、シンプルで素早く使えるTodoアプリです。

余計な機能を増やしすぎず、タスクの追加・完了・整理を簡単に行えることを重視しています。

## 主な機能

- Todoの追加・完了
- カテゴリ管理
- ゴミ箱機能
- 今日の達成画面
- 通知機能
- 通知設定

## 通知機能

- 昨日の達成数を通知
- アプリ未起動時のリマインダー
- 通知ON/OFF設定
- 通知時刻の設定

## 開発環境

- React Native
- Expo
- JavaScript

## プライバシーポリシー

Google Play提出には公開URLが必要です。
`privacy-policy.html` をGitHub PagesやGoogle Sitesなどで公開し、そのURLをGoogle Play Consoleの「アプリのコンテンツ」→「プライバシーポリシー」に登録してください。

## 広告

- バナー広告（画面下部）
- インタースティシャル広告（タスク5件完了ごと・5分間隔）
- リワード広告（カテゴリ枠を増やす時に視聴）
